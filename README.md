# Tomarigi

## 概要
Tomarigiは、ポートフォリオサイトとして制作しているWebサイトです。
デザインから実装まで自分で行い、HTML・CSS・JavaScriptを用いて開発しています。

## 制作目的
Webフロントエンドの学習成果を形にすることを目的として制作しました。
閲覧者に作品やプロフィールを分かりやすく伝えられる構成を意識しています。

## 使用技術
- HTML
- CSS
- JavaScript

## 主な実装
- JavaScriptによるUI操作
- CSSアニメーション
- GitHub Pagesで公開

## 工夫した点
デザインだけでなく、コードの保守性や可読性も意識しながら実装しています。
実際に動かして試し、改善を繰り返す開発スタイルで制作しています。
